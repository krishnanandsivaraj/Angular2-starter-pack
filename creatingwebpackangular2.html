What are model binders?

Model binders are helpful in mapping html forms ui elements to model. It acts as a bridge between html page and the model.

Model binders are used when we use html elements in the views. The name property for the html elements change that may requires changes to the model if model binders are not used. We cannot modify the models because these model s might be used in several places.

n order to do this we need to implement IModelBinder interface from the controller class. This IModelBinder class has BindModel method that accepts ControllerContext and ModeBinderContect class. A sample implementation for a form with two inputs namely the firstname and lastname will be like the below

public object BindModel(ControllerContext controllerContext, ModelBindingContext bindingContext) { HttpRequestBase httpRequest = controllerContext.HttpContext.Request; string firstName = httpRequest.Form.Get("firstname"); string lastName = httpRequest.Form.Get("lastname"); string age = httpRequest.Form.Get("Age"); return new Employee { Name = firstName + lastName,Age=age }; }

And the call for this in the controller will be

public ActionResult Index([ModelBinder(typeof(CustomHome))] Employee employee)
